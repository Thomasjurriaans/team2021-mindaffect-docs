text_field.py¶

This module is for text field management.

It contains a single class TextField.

A TextField has the following functionality:

  • Graphical text field for display

  • Logical text fields for value storage and keeping track of edits

  • Getters and setters for the logical text field

  • Add and undo function to edit logical text field

  • Replace last word function for logical text field

  • A boolean function is_empty() to check if the logical text field is empty

  • A function to update the suggestions
