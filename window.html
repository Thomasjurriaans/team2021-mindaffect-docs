window.pyÂ¶

This module is for window management.

It contains the superclass Window, and two subclasses Keypad and Menu.

A Window has the following functionality:

  • A get_keys() function to retrieve all keys

  • An activate() function to activate the window

  • A deactivate() function to deactivate the window

A Keypad has the following additional functionality:

  • A __build_keypad() function to set up the Keypad.

A Menu has the following additional functionality:

  • A __build_menu() function to set up the Menu.
