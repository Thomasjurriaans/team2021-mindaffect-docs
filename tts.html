tts module¶

This module is used to convert text into speech.

The module contains a single class Engine. Make an instance of this class in order to use it.

Usage:

import tts
tts = tts.Engine(connection_timeout=0.1, always_offline=False)
tts.speak("testing testing 123")

If there is a connection to the internet, Google’s gTTS will be used. If there is none, a local TTS service will be used by pyttsx3.

class tts.Engine(connection_timeout=0.1, always_offline=False)¶

Bases: object

Uses an online TTS service if available, local TTS otherwise.

Parameters

  • connection_timeout (float, optional) – slowest allowed connection speed in seconds.

  • always_offline (bool, optional) – if True, always use offline TTS.
