word_pred.py¶

This module implements GPT2 for word autocompletion and prediction.

The module contains a single class WordPred. Make an instance of this class in order to use it.

class word_pred.WordPred(settings_manager, update_ngram_depth=3, use_gpt=False)¶

Bases: object

This class autocompletes and predicts words

load_model()¶

Load the GPT2 model to use for prediction and autocompletion.

update()¶

Updates the language at use.

update_frequencies(text)¶

Updates the frequencies of n-grams in the n-grams file of the specified language.

Parameters

text – The text to update the n-gram frequency file with

predict(text='', n=3)¶

Gives n predictions based on the specified text. Either GPT2 or n-gram frequencies are used.

Parameters

  • text (str) – The context to base the predictions on (default is “”)

  • n (int) – The amount of predictions to return (default is 3)

Returns

A list of n predictions based on the specified context

freq_predict(front_ls, n, depth=3, all=False)¶

Gives n predictions based on the context and the language’s n-gram frequency file.

Parameters

  • front_ls (list(str)) – A list of words that specify the context

  • n (int) – The amount of predictions to return

  • depth (int) – The maximal n-gram size used for prediction (higher = more context used)

  • all (bool) – If true, instead of returning the best n predictions, it returns all predictions.

Returns

A list of n predictions based on the context and n-gram frequency file

Retrieves the predictions and their frequencies based on the frequency file.

Parameters

front (list(str)) – The first n-1 words from the n-gram that are used for prediction context

Returns

  • back (str) – The word that is predicted based on the specified front

Returns
